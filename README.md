## Descriptions

`My Mern Blog` was developed using [ReactJS](https://reactjs.org).

## Installation

To Install this Application :

- Clone  
  `git clone https://github.com/ivandi1980/my-mern-blog.git`
- Install
  ```bash
  yarn install
  ```

## Usage

To Start this Application, simply run

```bash
yarn start
```
